## üîß Recommended Settings

### Performance
- Allocate **6-8GB RAM** for optimal performance
- Use **Embeddium** settings to adjust visual quality vs performance
- Shaders are optional but included for those with capable hardware

### Voice Chat
- Server port: `24454` (UDP)
- Max voice distance: 48 blocks
- Whispering distance: 24 blocks
- Groups enabled for private conversations

### Waystones
- Generate naturally in villages and throughout the world
- Warp stone cooldown: 30 seconds
- Cross-dimensional travel enabled

---

## üéÆ Getting Started

1. **First spawn**: You'll receive an Animal Dictionary (Alex's Mobs) and Chronicle of Shadows (Apotheosis)
2. **Create a waystone network** for fast travel between bases
3. **Build a Tinkers' smeltery** for advanced tool crafting
4. **Explore the Twilight Forest** by throwing a diamond into water surrounded by flowers
5. **Set up automation** with Create or Applied Energistics 2

---

## ‚ö†Ô∏è Important Notes

- **Voice Chat** requires the server to have port 24454 open for UDP traffic
- **Lootr** makes all loot chests per-player, so everyone gets their own loot
- **Quark's Oddities** module is enabled, adding backpacks, magnets, and more
- All **YUNG's mods** are configured to replace vanilla structures
